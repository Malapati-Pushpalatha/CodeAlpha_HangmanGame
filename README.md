# CodeAlpha Hangman Game

## Description
This is a simple text-based Hangman game developed using Python as part of the CodeAlpha Internship.

## Features
- Random word selection
- Maximum 6 incorrect attempts
- Console-based user interaction

## How to Run
1. Install Python
2. Open terminal or command prompt
3. Run the program using:
   python hangman.py

## Tools Used
- Python
- PyCharm
